MY FAVORITE MOVIES
==================

Task:
Build a basic Rails app for curating a top-ten list of favorite movies from the TMDB API.


Requirements:
A user can search for a movie by title and add it to the list.
A user can remove a movie from the list.
A user can view their top-ten list.


Getting Started:
You can run the app locally or using Docker.

To start locally:
1. `bundle install`
2. `make start`

To start in Docker:
1. `make docker_build`
2. `make docker_start`
